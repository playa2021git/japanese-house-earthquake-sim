# japanese-house-earthquake-sim

単一HTMLファイルで完結する、日本家屋の軸組（柱・梁・筋交い）の耐震性を見せる教育用の疑似物理シミュレーターです。

## 使い方

`index.html` をブラウザで直接開くと動作します。外部ライブラリ、外部CSS、外部画像、ビルドツールは不要です。

## 教材のポイント

- 柱と梁だけの四角い架構は、横揺れで平行四辺形のようにラッキング変形しやすいことを示します。
- 片筋交いは筋交いなしより変形しにくい一方、圧縮方向では座屈しやすく不安定になることを示します。
- X字筋交いは左右両方向の揺れに対応しやすく、最も変形しにくいことを示します。
- 部材の負荷を緑→黄→赤で表示し、限界を超えた部材は灰色の破断済みとして表示します。

## 実装方式

Vanilla JavaScript と Canvas 2D だけで、Verlet積分 + スティック拘束による Position Based Dynamics 風の疑似物理を実装しています。
