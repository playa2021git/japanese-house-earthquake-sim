# japanese-house-earthquake-sim

単一HTMLファイルで完結する、日本家屋の軸組（柱・梁・筋交い）の耐震性を見せる教育用の疑似物理シミュレーターです。

## 使い方

`index.html` をブラウザで直接開くと動作します。外部ライブラリ、外部CSS、外部画像、ビルドツールは不要です。

## v0.4 のポイント

- 二階建て・4列・3スパンの日本家屋風軸組模型に整理しました。
- ノードに `role` / `floor` / `col` / `editable` を持たせ、groundAnchor と ridgeNode は筋交い編集対象から除外しています。
- スティックに `floor` / `span` / `direction` / `userAdded` / `damage` を持たせ、1階・2階・左右偏り・筋交い方向を評価できるようにしました。
- 筋交いは、1つの階・1つのスパンの対角線だけに追加できます。
- 1階層間変形、2階層間変形、1階ダメージ、2階ダメージを別々に計算・表示します。
- 筋交い本数だけでなく、1階の弱さ、2階だけ補強、左右の偏り、片方向偏り、X字の有無、上下階のズレを評価します。
- 初期震度は震度4です。震度1〜7は教材用の簡易プリセットで、実際の地震被害を厳密に再現するものではありません。

## 教材のポイント

- 柱・梁だけの二階建て軸組は、特に1階が弱いと横揺れで大きく変形しやすいことを示します。
- 2階だけ、または片側だけ筋交いを入れても、強い構造とは限らないことを示します。
- 1階と2階に左右バランスよくX字筋交いを入れると、層間変形とダメージが減りやすいことを示します。
- 震度1でも長時間加振で危険になり得る表現は、弱い教材模型と繰り返し揺れの影響を学ぶための簡易表現です。

## 実装方式

Vanilla JavaScript と Canvas 2D だけで、Verlet積分 + スティック拘束による Position Based Dynamics 風の疑似物理を実装しています。有限要素法や建築基準法の厳密な壁倍率計算ではなく、教育用に比較の方向性が分かる疑似物理です。
