# japanese-house-earthquake-sim

単一HTMLファイルで完結する、日本家屋の軸組（柱・梁・筋交い）の耐震性を見せる教育用の疑似物理シミュレーターです。

## 使い方

`index.html` をブラウザで直接開くと動作します。外部ライブラリ、外部CSS、外部画像、ビルドツールは不要です。

## v0.3 のポイント

- 初期状態は「耐力壁なし・筋交いなし」の弱い日本家屋です。
- 白い接点（柱脚・柱頭）を2か所クリックして、1スパン内の対角線に筋交いを追加できます。
- 筋交い追加モード、1本取り消し、全消去、おすすめ配置の自動追加を備えています。
- 震度1〜7を選べます。初期値は震度3です。
- 構造部材に累積ダメージを持たせ、低震度でも筋交いなしで長時間揺らすと危険になりやすい教材表現にしています。
- 部材を太さのあるポリゴンとして描画し、接合部、加振台、警告ランプ、応力発光、破断断面を表示します。

## 教材のポイント

- 耐力壁なし・筋交いなしの木造軸組は、横揺れで平行四辺形のようにラッキング変形しやすいことを示します。
- 生徒が自分で筋交いを入れて、どの配置が変形を抑えるか試せます。
- バランスよく筋交いを入れると左右両方向の揺れに対応しやすく、最も変形しにくいことを示します。
- 部材の負荷を緑→黄→赤で表示し、累積ダメージと破断で弱い構造の危険性を見せます。

## 実装方式

Vanilla JavaScript と Canvas 2D だけで、Verlet積分 + スティック拘束による Position Based Dynamics 風の疑似物理を実装しています。震度は教材用の簡易表現であり、実際の地震動を厳密に再現するものではありません。
