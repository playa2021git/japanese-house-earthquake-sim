# japanese-house-earthquake-sim

単一HTMLファイルで完結する、日本家屋の軸組（柱・梁・筋交い）の耐震性を見せる教育用の疑似物理シミュレーターです。

## 使い方

`index.html` をブラウザで直接開くと動作します。外部ライブラリ、外部CSS、外部画像、ビルドツールは不要です。

## v0.5 のポイント

- v0.4 の二階建て・4列・3スパン模型を維持しつつ、GUI・物理値・評価ロジックの整合性を検証しやすい設計に整理しました。
- ノードに `mass` / `invMass` を持たせ、拘束解決で屋根・2階床などの重さを反映します。
- 震度プリセットを再校正し、震度1〜7の振幅・周波数・慣性・ダメージ率を教材用の簡易値として扱います。
- 1階層間変形、2階層間変形、最大負荷、1階ダメージ、2階ダメージを同じ内部指標から表示・判定します。
- 配置評価は、1階の弱さ、2階だけ補強、左右偏り、片方向偏り、X字、上下階の補強位置ズレを storyDamage に反映します。
- 「内部テスト」ボタンで、筋交いなし、2階だけ補強、片側だけ補強、良配置、震度7、震度1長時間加振の代表ケースを実行できます。
- v0.5復旧版では、起動時のDOM不一致でCanvasが空白にならないように、内部テストボタンのHTML/JS整合、null安全なイベント登録、起動エラー表示、構造生成検査を追加しました。
- Canvas 2D のまま、弱い階のハイライト、層間変形ゲージ、ポリゴン部材、発光、破断断面、加振台表示を行います。

## 教材のポイント

- 柱・梁だけの二階建て軸組は、特に1階が弱いと横揺れで大きく変形しやすいことを示します。
- 2階だけ、または片側だけ筋交いを入れても、強い構造とは限らないことを示します。
- 1階と2階に左右バランスよくX字筋交いを入れると、層間変形とダメージが減りやすいことを示します。
- 震度1でも長時間加振で危険になり得る表現は、弱い教材模型と繰り返し揺れの影響を学ぶための簡易表現です。

## 実装方式

Vanilla JavaScript と Canvas 2D だけで、Verlet積分 + 質量付きスティック拘束による Position Based Dynamics 風の疑似物理を実装しています。有限要素法や建築基準法の厳密な壁倍率計算ではなく、教育用に比較の方向性が分かる疑似物理です。
